Предмет практикум на ЭВМ подразумевает программирование численных методов математической физики. Здесь описаны технические аспекты, связанные с подготовкой ПК (установка и минимальное введение).

1. Код пишем на языке python 3.
2. Отчеты выполняем в latex.


<h3> Учим python </h3>
Могу порекомендовать online сервис для начального обучения
<a href='http://pythontutor.ru/'> Питонтьютор </a>
Курс содержит около 128 задач самого простого уровня, которые можно сделать за неделю активной работы. При этом даже ставить ничего не нужно - всё запускается в браузере.


<h3> Настраиваем python в Ubuntu </h3>
Если у Вас установлена Ubuntu 16.04 и выше, то интерпретатор уже есть. Убедиться в этом можно, запустив его в терминале. Далее все команды, которые вводятся в терминале начинаются с символа $. Сам символ вводить не нужно!
<code>
$ python3
</code>
Завершить сессию можно комбинацией Ctrl + D.

Для отрисовки графики нужна библиотека matplotlib, которую поставим через pip - менеджер пакетов для python:
<code>
$ sudo apt install python3-pip
</code>

Ставим библиотеку matplotlib:
<code>
$ sudo pip3 install matplotlib
</code>

Теперь убедимся, что картинки рисуются. Подключим класс pylab из matplotlib (и обзовем его plt). Далее подготовим список значений, которые хотим отрисовать. Добавим соответствующие точки на плоскость методом plot() и посмотрим на результат методом show().
<code>
$ python3
from matplotlib import pylab as plt
y = [1, 5, 2, 4, 3]
plt.plot(y)
plt.show()
</code>

Чтобы не работать в интерпретаторе, наберем команды в файле с помощью любого текстового редактора (gedit, nano, vim и далее) и сохраним (например, под названием 'sem_0.py'). Далее запустить выполнение можно командой:
<code>
$ python3 sem_0.py
</code>

# Начальные условия $u_0(x)$
def u0(x):
	return 1.0 - (x - 0.5)**2

# Граничные условия $\varphi_1(x)$, $\varphi_2(x)$
def phi1(t):
	return 0.0
def phi2(t):
	return 0.0

# Параметры задачи: $\mu$ - коэффициент теплопроводности, $T$ - время искомого распределения температуры
mu = 0.1
T = 1.0

# Сетка по пространству
n = 10
h = 1.0 / n
x = [i * h for i in range(0, n + 1)]

# Сетка по времени
m = 50
tau = T / m
t = [k * tau for k in range(0, m + 1)]

# Условие устойчивости
r = 2 * mu * tau / h / h
print(tau, h, r)

# Инициализация начального слоя
y = [u0(x[i]) for i in range(0, n + 1)]
for i in range(0, n + 1):
	y[i] = u0(x[i])

# Счет
ynew = [0] * (n + 1)
for k in range(0, m):
	y[0] = phi1(t[k])
	y[n] = phi2(t[k])
	for i in range(1, n):
		ynew[i] = y[i] + tau * mu / h / h * (y[i+1] - 2 * y[i] + y[i-1])
	y = list(ynew)

# Вывод
print(x)
print(y)

# График
from matplotlib import pylab as plt
plt.plot(x, y)
plt.show()
</code>

<h3> Настраиваем jupyter в Ubuntu </h3>
Если Вы хотите, чтобы вся работа интерпретировалась в браузере, то следует установить отдельный пакет для веб-интерфейса
<code>
$ sudo apt install jupyter
</code>

Теперь, чтобы запутить локальный сервер с интерпетатором
<code>
$ jupyter notebook
</code>
Терминал на время работы закрывать нельзя.

В открывшемся бразуере будет запущена страница localhost:8888/tree, которая показывает структуру в домашней директории. Зайдем в соответствующее место на диске и создадим так называемый notebook. Нажимаем New в правом верхнем углу и выбираем Python 3. Теперь можем набирать описание (markup) или код (code) в виде блоков. 
<code>
		Описание

In [1]  2+3
</code>


Для ввода есть 2 режима: набора и управления (прям как в знаменитом vim). 
Влючить режим набора: enter. 
Включить режим ввода: escape. 
Посмотреть все горячие клавиши: h. 
Добавить блок сверху: a.
Добавить блок снизу: b.
Сделать блок описанием: m.
Сделать блок кодом: y.
Сделать заголовок: 1. 
Запустить и перейти ниже: shift + enter.
Запустить и остаться: ctrl + enter.

Данная штука дает две интересных возможности - удобная отладка и сопровождение читаемым текстом с поддержкой LaTeX.
