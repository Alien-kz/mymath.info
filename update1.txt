В раздел <a href='abiturient'>олимпиады по математике</a> добавлены материалы:

+++ 219 задач для подготовки к IMC-2018. Данные материалы были использованы для подготовки команды Казахстанского филиала МГУ на IMC-2018, где ребята выступили очень успешно: Бекмаганбетов Бекарыс (ММ-2) взял золото (6 задач из 10), Аскергалиев Ануар (ВМК-2) взял серебро (4 задачи из 10), а Журавская Александра (ВМК-4) получила бронзовую награду (2 задачи из 10). В брошюре собраны задачи из различных источников: задачи IMC, задачи Putnam, Putnam And Beyond (Andreescu), задачи студенческих олимпиад (Садовничий, Григорьян, Конягин) и другие источники. <a href='http://127.0.0.1/math/index.php?olymp=msu&year=2018-imctrain'> 219 задач для подготовки к IMC-2018 </a>.
